# WoTStatistics
Unofficial bot for discord to show the statistic of player in World of Tanks
## What's technlogy use?
- Wargaming API
- DSharpPlus
- C#
## How to use it?
1. Add bot to discord.
2. In discord, you can use this commands:
   - ```&stats your_nickname```
   - ```&help```
